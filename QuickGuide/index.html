The TPTP Problem Library

The TPTP (Thousands of Problems for Theorem Provers) is a library of test problems for ATP systems.
  • The WWW page is tptp.org/TPTP
  • You can browse the TPTP problems online.

The TSTP Solution Library

The TSTP (Thousands of Solutions from Theorem Provers) is a library of solutions to TPTP problems.
  • The WWW page is tptp.org/TSTP.
  • You can browse the TSTP solutions online.
  • See E's (and others') derivation (a refutation) for the FOF problem PUZ001+1

Test Problems

The SYN000* problem files are contrived to use most features of the TPTP language, and are thus suitable for testing parsers, etc.
  • CNF problem contrived to use common parts of the TPTP language: SYN000-1
  • FOF problem contrived to use common parts of the TPTP language: SYN000+1
  • TFF problem contrived to use common parts of the TPTP language: SYN000_1
  • THF problem contrived to use common parts of the TPTP language: SYN000^1
  • TFF problem contrived to use non-classical parts of the TPTP language: SYN000_6
  • THF problem contrived to use non-classical parts of the TPTP language: SYN000^6
